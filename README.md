# Авто Видео

Быстрый локальный видеоплеер для автомобильных Android-магнитол и Android-планшетов. Приложение работает без интернета и читает медиафайлы из внутренней памяти, USB-накопителей, SD-карт и выбранных пользователем папок.

## Основные возможности

- быстрый поиск видео и музыки через MediaStore;
- ограниченный по времени резервный поиск в файловой системе;
- приоритетная проверка папок Movies, Video, Download, DCIM и Music;
- постоянный кэш найденной медиатеки между запусками приложения;
- повторное открытие приложения без полного сканирования накопителей;
- главная страница только с папками, содержащими видео;
- отдельные папки с видео и отдельные папки с музыкой;
- раздел «Продолжить просмотр» внутри вкладки «Видео»;
- кнопка быстрого продолжения последнего видео в боковом меню;
- воспроизведение через LibVLC с программной поддержкой AC3, E-AC3 и DTS;
- принудительный стереовыход без цифрового passthrough;
- восстановление позиции просмотра для каждого файла;
- перемотка кнопками ±10 секунд, двойным касанием, ползунком и горизонтальным свайпом;
- переход к предыдущему и следующему файлу;
- восстановление видеоповерхности после возврата приложения из фона;
- крупные элементы управления для автомобильной магнитолы;
- плавные slide/fade-переходы и анимации нажатий;
- часы на всех основных экранах;
- системная кнопка перехода к разрешениям приложения;
- двойной жест «Назад» для закрытия приложения.

## Поддерживаемые файлы

Сканер распознаёт MP4, M4V, MKV, WEBM, AVI, MOV, TS, M2TS, MTS, MPG, MPEG, 3GP, FLV, WMV, VOB, OGV, MP3, M4A, AAC, FLAC, WAV, OGG, OPUS, AMR, AC3, E-AC3, DTS, WMA, APE, ALAC и AIFF.

Фактическая совместимость зависит от видеокодека, разрешения, битрейта и производительности головного устройства. Для звука используется программный движок LibVLC, а для видео по возможности включается аппаратное декодирование.

## Как пользоваться

1. При первом запуске разрешить доступ к видео, музыке и памяти.
2. Для максимального доступа включить системное разрешение «Управление всеми файлами».
3. Подключить флешку, SD-карту или внешний диск.
4. Дождаться первого индексирования медиатеки.
5. При необходимости добавить отдельную папку через «Настройки» → «Добавить носитель».
6. При последующих запусках ранее найденные папки загружаются из кэша практически сразу.

## Сборка

Требования:

- Android Studio и JDK 17;
- Android SDK 36;
- Gradle 8.13.

Команда:

```bash
gradle :app:assembleDebug
```

APK создаётся по пути:

```text
app/build/outputs/apk/debug/app-debug.apk
```

## Архитектура APK

Версия 0.6.0 собирается только для `arm64-v8a`. Это уменьшает размер APK примерно с 200+ МБ до 70 МБ и соответствует большинству современных Android-магнитол и планшетов. Для 32-битных устройств потребуется отдельная сборка `armeabi-v7a`.
