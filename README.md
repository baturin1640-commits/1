# Авто Видео

Приложение для автомобильного головного устройства: просмотр видео и прослушивание аудио со съёмных носителей без интернета.

## Что уже реализовано

- подключение нескольких флешек, SD-карт и внешних дисков через системный выбор папки Android;
- сохранение доступа к выбранным носителям после перезапуска приложения;
- рекурсивный поиск медиафайлов во вложенных папках;
- на главном экране показываются только папки, содержащие видео;
- отдельные разделы со всеми видео и всеми аудиофайлами;
- воспроизведение через AndroidX Media3 / ExoPlayer;
- восстановление позиции просмотра для каждого файла;
- перемотка тремя способами:
  1. кнопки «−10 секунд» и «+10 секунд»;
  2. двойное касание по левой или правой половине экрана;
  3. свободная перемотка ползунком;
- горизонтальная полноэкранная компоновка для автомобильного ГУ;
- тёмный интерфейс с фиолетовыми акцентами по выбранному дизайну;
- автоматическая сборка debug APK в GitHub Actions.

## Поддерживаемые файлы

Сканер распознаёт распространённые контейнеры и расширения, включая MP4, MKV, WEBM, AVI, MOV, MPEG, TS/M2TS, MP3, AAC, M4A, FLAC, WAV, OGG, OPUS, AC3, EAC3, DTS, WMA, APE и ALAC.

Фактическое декодирование конкретного файла зависит от кодека внутри контейнера и декодеров, доступных на головном устройстве. При неподдерживаемом кодеке плеер показывает понятное сообщение об ошибке. Для редких кодеков следующим этапом можно подключить программный FFmpeg/LibVLC-движок.

## Как пользоваться

1. Подключить флешку или внешний диск к головному устройству.
2. Открыть приложение и нажать **«Выбрать носитель»**.
3. В системном окне Android выбрать корневую папку носителя и подтвердить доступ.
4. Дождаться сканирования — папки с видео появятся на главном экране.
5. Для второго носителя открыть **«Настройки» → «Добавить носитель»**.

## Сборка

Требования:

- Android Studio с JDK 17;
- Android SDK 36;
- Gradle 8.13.

Команда сборки:

```bash
gradle :app:assembleDebug
```

Готовый APK появляется по пути:

```text
app/build/outputs/apk/debug/app-debug.apk
```
